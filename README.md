# Loan Default Prediction System

### Project Overview
- The project implements **End to End Machine Learning System** that predicts the probability of **Loan Default** for applicants in real time. 

### Problem Definition 
- **Financial Insititutions** faces significant losses when a borrower default on loan. This **reduces profits and increases Operational Costs** such as acquiring debt collection agencies

### Solution 
- To build a model that will predict whether the applicant will default the loan or not. From this, financial institutions can **identify high risk applicants** before loan approval and make decisions which helps in reducing the **credit loss, operational cost** and **Non-Performing Asset (NPA)**.
